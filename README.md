# Library Management System

A full CRUD Library Management System built with PHP and MySQL.  
The system allows users to manage books, mark favorites, search records, and export data.

## Features

- User Authentication (Register / Login / Logout)
- Add Books
- Edit Books
- Delete Books
- View Books
- Search by title, author, or year
- Favorites system
- Pagination
- Book cover upload
- Export data to CSV
- Dark / Light mode UI

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## How to Run

1 Install XAMPP

2 Move project folder to:

C:\xampp\htdocs\library-management-system

3 Start Apache and MySQL

4 Create database:
library_db

5 Import tables

6 Open browser:

http://localhost/library-management-system
